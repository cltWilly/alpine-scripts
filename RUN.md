### Run
Clone the script

```bash
  wget -O run_and_delete.sh https://raw.githubusercontent.com/cltWilly/alpine-scripts/main/run-and-delete.sh
```

mod the sh to execute
```bash
chmod +x run_and_delete.sh
```

run the script

```bash
./run_and_delete.sh
```
remove the script after done
```bash
rm -f run_and_delete.sh
```
